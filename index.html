About This Project

SkyWatch is a cloud-based Web GIS platform for monitoring and predicting seabird migration patterns in the Asia-Pacific region, with a focus on the critically endangered Chinese Crested Tern (Thalasseus bernsteini).

The platform integrates satellite-derived environmental layers (Chlorophyll-a, SST, mangrove extent) with citizen science observations to perform Multi-Criteria Analysis (MCA) and identify optimal locations for artificial floating habitat platforms.

Data Sources
  • Movebank — tern migration routes
  • Global Mangrove Watch (GMW) — habitat boundaries
  • MODIS/VIIRS via GEE — Chlorophyll-a, SST
  • Global Fishing Watch — shipping density
  • Community submissions (this platform)
Tech stack: Mapbox GL JS · PostGIS / Supabase · GeoServer · Google Earth Engine · QGIS
